インボイス制度対応

適格請求書発行事業者の登録番号(T番号)、税率ごとの区分記載(10%/軽減8%/対象外)、軽減税率の※印まで自動。

源泉徴収の自動計算

デザイン料・原稿料などの報酬請求に。10.21%、100万円超の部分は20.42%の二段階計算に対応。

完全オフライン

処理はすべて端末内。アカウント登録もサーバー送信もなし。取引先情報が外部に出る心配がありません。

柔軟な税計算

外税/内税の切替、端数処理(切り捨て/四捨五入/切り上げ)を選択可能。

取引先プリセット

取引先ごとに保存してワンクリック切替。JSON書き出しでバックアップ・PC移行も簡単。
